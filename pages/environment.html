A. Open Files from the Terminal


Depending on what operating system you are using, there are different installation steps you may need to take to get this functionality working. Most of us will be using macOS or Windows machines. Here are the links to the documentation if you need more information about the setup process:

macOS - https://code.visualstudio.com/docs/setup/mac

Windows - https://code.visualstudio.com/docs/setup/windows (recommend using the installer)


NOTE: Windows users should have the functionality ready to go. If you are a macOS user, then follow the next series of steps:

  • Open Visual Studio Code.
  • Press command + shift + p to open the command palette.
  • Type in shell command.
  • Select the option: "Install 'code' command in PATH".
  • Restart your terminal if it is open.

Once the shell command has been installed you can now use the code command in your terminal to open individual files or entire folder structures straight from the terminal! Here are the two commonly used options.

  • code (file or folder name)
  • code .

I highly recommend using the second option: code . which works by navigating (in terminal or command line of course) to and INTO the target FOLDER. For example, I will be opening my GitHub Pages project folder:
